Lists4Home Introduction and Operation

Back
Lists4Home is a stupid name I came up with for what is basically (at this point) a shopping list. It is a web based application which can be used to share shopping lists between a group of people. The website has been designed to work on any device – especially cell phones.

Basic Usage

  • Create one user account which will be used by all parties. (i.e. one account will be used by the husband, wife, girlfriend, boyfriend etc)
  • On android and apple phones you can create a bookmark in your browser and make a shortcut on the main screen. I suggest doing this to access the list easily.
  • For android go to the website in the phones browser and create a bookmark. Then add this bookmark to the home screen.

Instructions

Going to the link will take you to this page:

You can ignore everything here (feel free to look) but click on Shopping List. Assuming you have never registered before, you will be presented with this screen.

Click on “Click Here to Register” and This screen will emerge:

Fill everything in as per normal and click Submit. A message will come up signifying a successful user creation. At this point, log in using the details you just set up. You should only ever need to log on once on a given device. This screen will come up after logging in.

Add Items – add items onto the shopping list. Remove Items – removes items from the shopping list. Bought Recently- Clicking on this shows the last seven days purchases (i.e. items that have been removed from the list). Clicking on Add Items brings up this screen:

Start typing an item into the Item field and a list of predefined items will be listed. Choose the desired item and click submit. If the item is not on the list when you click Submit the system will ask you to categorise this “new” item. Choose the category that best suits the new item.

The field “Mesaurement,Quantity, etc” gives you the opportunity to add free text to your item. For instance if you need 36 eggs (for the mother of all crème brulees or something) you would enter Eggs in the Item field and 36 in the measurement field. For example:

After clicking Submit Items the screen will go back to the list.

This is your list. While you are shopping you can click on eggs once you have them.

If you then click on Remove Items this item will be removed from the list.
